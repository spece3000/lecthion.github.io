NODE.JS

node.js - это среда выполнения js, которая работает на сервере

Приложение Node.js выполняется одним процессом, без создания нового потока для каждого запроса

Чтобы запустить фрагмент Node.js, сохраните его как файл server.js и запустите node server.js в терминале.

  1. Этот код сначала включает в себя модуль Node.js http.
  2. Метод createServer() из http создает новый HTTP-сервер и возвращает его.

Cвойство statusCode - 200, указывает на успешный ответ.

SetHeader устанавливает заголовок для сайта

Мы завершаем закрытие ответа, добавляя содержимое в качестве аргумента к end()

Node.js фреймворки и инструменты

  • Express - Минималистичный фреймворк для сервера
  • Next.js - Фреймворк для рендеринга приложений на стороне сервера на React
  • Meteor - Мощный фреймворк для разработки различных приложений
  • Socket.io - Это механизм коммуникации в реальном времени для создания сетевых приложений.
  • Koa - Созданный той же командой, что и Express, Koa стремится быть еще проще и меньше, опираясь на многолетние знания.

V8 - это название движка JavaScript, на котором работает Google Chrome. Именно он принимает наш JavaScript и выполняет его во время просмотра веб-страниц в Chrome.
